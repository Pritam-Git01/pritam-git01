## About me

Full Stack Developer & Software Engineer based in Bengaluru, India. I work mostly with React, Next.js, Node.js, TypeScript, and Python across web and cloud projects. Currently focused on building AI-powered products, RAG systems, and multi-agent conversational AI.

Day-to-day I build full-stack web applications, integrate AI/LLM features into products, and design scalable systems - it's a core part of what I do both professionally and in my own ventures.

2.11+ years of experience across React, Node.js, Python, TypeScript, MongoDB, PostgreSQL, AWS, Docker, and Kubernetes. Available for full-time, contract, and freelance opportunities.

## What I'm working on (personal)

- **[BuildPilot AI](https://github.com/pritam-git01)** - AI-powered product intelligence tool for product managers, founders, and engineers. React, Node.js, LLMs, RAG.
- **[Crypto Wallet & Exchange Platform](https://github.com/pritam-git01/crypto-wallet)** - A full-featured crypto wallet and exchange platform with secure wallet management, token swapping, portfolio tracking, and real-time market data. React, Next.js, RainbowKit, Tailwind, recharts.
- **[AI-Powered RFP Management](https://github.com/pritam-git01/rfp-management)** - Full-stack RFP management platform using AI to streamline procurement, from natural-language RFP creation to automated vendor proposal parsing. React, Next.js, OpenAI API, Node.js, MongoDB, Gemini.
- **[Pipeline Builder](https://github.com/pritam-git01/pipeline-builder)** - Visual drag-and-drop pipeline builder for designing and validating data workflow DAGs. React Flow, FastAPI, Next.js, Shadcn UI.
- **[JSON Tree Visualizer](https://github.com/pritam-git01/json-tree-visualizer)** - Interactive tree visualizer for exploring and manipulating complex JSON data structures. React, TypeScript, Tailwind CSS, Reactflow.
